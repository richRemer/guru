Guru Package Manager
====================

Install
-------
The Guru Package Manager uses the included **guru** tool to install the package
manager.

```sh
git clone https://github.com/richRemer/guru.git
cd guru
bin/guru install
```

At the end of the install you will see "completed installing guru" to know the
software has been installed.

More Help
---------
Once you have installed the Guru Package Manager, you can use the included man
page to get more help on using the tool.

```sh
man guru
```
